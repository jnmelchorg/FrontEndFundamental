Tutorial

Chapters in video

Chapter 1
Covers the basics to get started with html. It describes at high level the type of html DOCTYPE, setting the language for the page and basic CSS to make the page dark.
Chapter 2
Covers the basics of the head element. It describes basic metadata and links. Also, he explains that anything in head is not shown in the webpage
Chapter 3
Covers the basics of the body element. It describes headers, paragraphs, horizontal rulers, emphasis (italic) and strong (bold), spacing, abbreviations, addresses, logical symbols and comments
Chapter 4
Covers the basics of lists. Three types of lists are explored:
  1. Unordered lists
  2. Ordered lists
  3. Description Lists
The following link leads to the html page for the list part: Lists page.
Chapter 5
Covers the basics of links. It covers the anchor tag and how to use in the following way:
  • Reference another website on the internet through absolute references.
  • Reference a webpage in the local server through relative references.
  • Reference to an area within the same page, which the author of the video calls internal reference.
  • Changing colours of links
  • Download links
  • Telephone links
  • Target links for absolute references to other websites to open them in a new tab
  • Reference to the previous file. The example is in Lists page
Chapter 6
Covers the basics of adding images to the webpage. The training webpage is Images. This training includes the image tag, and the basics of using it by providing the source and alt. Furthermore, providing the width and height is important for the webpage and finally the loading option defines how the imagines should be loaded, with eager downloading the image for the page immediately and lazy downloading images in demand and just when getting close to the image in the webpage. This section also covers the figure tag and the figcaption tag. The figure attribute can contain code.
Chapter 7
This chapter covers the basics of semantic tags in html. The training webpage is Semantic Tags. In the file the tags: header, main and footer. The chapter also cover articles, which are an alternative to sections. The chapter also covers the mark, time, details and summary tags. Finally the video highlights that the div and span should be avoided in html but can be used in CSS.
Chapter 8
This chapter covers the basics of tables. The training webpage is Tables.
